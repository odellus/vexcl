template<class Clock = std::chrono::high_resolution_clock>
class vex::profiler< Clock >

Class for gathering and printing OpenCL and Host profiling info.

Parameters
Clockclass that provides interface compatible with either boost::chrono or std::chrono clocks.
AvgKind of averaging used. AvgMedian is more stable, but may have high overhead when profiling is made in a loop.
